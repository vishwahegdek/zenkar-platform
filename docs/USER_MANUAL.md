# Zenkar Platform - User Manual

Welcome to the Zenkar Platform User Manual. This guide covers all the features available to you for managing orders, products, and customers.

## 1. Getting Started

### Accessing the Platform
1.  Navigate to the application URL (e.g., `https://order.zenkar.in`).
2.  You will see the Login screen.
3.  Click **"Login with Google"**.
4.  Select your authorized Google account.
5.  Upon successful login, you will be redirected to the **Dashboard**.

## 2. Dashboard
The Dashboard provides a quick overview of the business.
*   **Recent Orders**: View the latest orders placed.
*   **Quick Actions**: Shortcuts to create new orders or products.

## 3. Order Management

### Viewing Orders
*   Click **"Orders"** in the sidebar.
*   You will see a list of all orders.
*   **Search**: Use the search bar to find orders by Customer Name or Order Number.
*   **Filter**: Filter orders by status (e.g., "Confirmed", "Production", "Delivered").

### Creating a New Order
1.  Click **"New Order"** or the **"+"** button generally found in the header.
2.  **Customer**: Select an existing customer or create a new one on the fly.
3.  **Products**:
    *   Add products to the order.
    *   Adjust quantity and unit price.
    *   Upload images for specific customizations (if needed).
4.  **Dates**: Set "Order Date", "Due Date", and "Delivery Date".
5.  **Payment**: You can record an advance payment immediately.
6.  Click **"Save Order"**.

### Editing an Order
1.  Click on any order in the "Orders List".
2.  Click the **"Edit"** button (pencil icon).
3.  Make necessary changes to items, dates, or notes.
4.  Click **"Update"**.

### Adding Payments
1.  Open an Order.
2.  Scroll to the **"Payments"** section.
3.  Click **"Add Payment"**.
4.  Enter the Amount, Date, and a Note (e.g., "UPI Ref: 123456").
5.  Save. The "Balance Due" will update automatically.

## 4. Product Management

### Product Catalog
*   Click **"Products"** in the sidebar.
*   View all available products with their default prices.

### Adding a Product
1.  Click **"New Product"**.
2.  **Name**: Enter product name.
3.  **Default Price**: Set the base price.
4.  **Image**: Upload a product image.
5.  **Notes**: Add any internal notes.
6.  Save.

## 5. Customer Management

### Customer Directory
*   Click **"Customers"** in the sidebar.
*   View a list of all customers with contact details.

### Adding a Customer
1.  Click **"New Customer"**.
2.  Enter **Name**, **Phone**, and **Address**.
3.  Save.
